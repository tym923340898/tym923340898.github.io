逻辑运算符（&|单个双个区别）

逻辑运算符 描述
&& 一旦运算除左端为false,那么右端就不运算了
& 运算两端都是true,结果才是true,那么只要有任意一端false,结果为false
|| 一旦运算左端为true,结果就为true,都断不用计算
| 运算两端任意一段为true,结果为true,只有两端都为false,结果为false
